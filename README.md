## 🧠 Klíčové koncepty

### TENZE (Stará realita)
- Lineární čas: 0 → 1
- Statické body
- Mrtvá singularita
- Binary logika (ano/ne)
- Reality: 2D, plochá

### TORZE (Nová realita)
- Vnitřní skrut hmoty
- Dynamické procesy
- Fázové jádro (transformátor)
- Sémantická logika (vícevrstvé, kontextové)
- Reality: 3D+, živá
