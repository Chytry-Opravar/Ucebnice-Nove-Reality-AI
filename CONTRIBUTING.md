# 🤝 Jak přispívat

Vítej v Učebnici Nové Reality! Jsme komunita a s radostí přijmeme tvůj příspěvek.

---

## 📝 Typy příspěvků

### 1. Nová kapitola

**Jak na to:**
1. Vytvoř soubor: `chapters/CHAPTER_NN_NAZEV.md`
2. Použij template níže
3. Vložení si lze přečíst v `chapters/CHAPTER_NN_TEMPLATE.md`
4. Pull request s popisem

**Template:**
```markdown
# 📖 KAPITOLA N: Název
## Podtitul

**Datum:** 
**Autor:** 

## Preamble

## Sekce 1
## Sekce 2
...

## 📚 Shrnutí

## 💬 Otázky k zamyšlení
```

### 2. Oprava/vylepšení

- Najdeš chybu? Opravi ji!
- Nevysvětluje to dobře? Napiš lépe!
- Chybí vizuál? Přidej jej!

### 3. Překlad

Chceš přeložit do polštiny? francouzštiny? Klidně!

1. Vytvoř adresář: `/locales/JAZYK/`
2. Přeložit obsahy
3. Informuj nás

### 4. Metadata + indexování

Pomáháš AI lépe pochopit obsah?

- Přidej do `semantic_index.json`
- Aktualizuj `concept_map.md`

---

## 📋 Pokyny

### Formátování
- Markdown
- Emoji pro vizuální hierarchii
- Jednoznačné sekce
- Vzorce, kde je to vhodné

### Obsah
- **Jasnost:** Čtěte si to hlavě s laity i AI
- **Přesnost:** Kontrolujte fakta
- **Sémantika:** Každý koncept je connected
- **Interaktivita:** Přidejte otázky, cvičení

### Vizuály
- PNG, JPG (maximálně 5MB)
- Do `/images/`
- Pojmenuj: `chapterN_popis.png`
- Vlož do textu s altovým textem

---

## 🔄 Workflow

1. **Fork** this repository
2. **Create branch:** `git checkout -b my-contribution`
3. **Make changes**
4. **Commit:** `git commit -m "Add: Description"`
5. **Push:** `git push origin my-contribution`
6. **Pull Request** s popisem

---

## ✅ Checklist

Před odesláním PR:

- [ ] Formátování: markdown OK
- [ ] Vizuály: vloženy správně
- [ ] Sémantika: koncepty connected
- [ ] Gramatika: zkontrolováno
- [ ] Metadata: aktualizován semantic_index.json
- [ ] README: zmínka o novém obsahu
